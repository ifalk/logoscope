Le fichier .csc doit être placé dans le répertoire "Cassys" de votre dossier de travail Unitex et édité afin de pointer vers l'adresse de votre dossier de travail personnel. 
Le répertoire Logoscope doit être placé dans le répertoire "Graphs".
Les dictionnaires doivent être placé dans le répertoire "Dela" et être appliqués au texte avant d'appeler la cascade afin d'obtenir les meilleurs résultats. 

Le dernier transducteur de la cascade est utilisé afin de tenter d'augmenter le rappel de l'outil au prix d'une baisse de la précision. Pour obtenir les résultats les plus précis possibles
il est conseillé de le désactiver par l'interface graphique d'Unitex ou en supprimant la dernière ligne du fichier .csc
